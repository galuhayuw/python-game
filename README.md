# Case Study: North America Video Games Sales Recovery Strategy (2017)

## Project Overview & Problem Statement
Industri video game di Amerika Utara (NA) mengalami penurunan drastis sebesar 56,05% sejak tahun 2012. Tahun 2016 mencatatkan performa terendah dengan penjualan hanya $44,93 Juta.
Tujuan Utama: Merumuskan strategi berbasis data untuk meningkatkan penjualan di Amerika Utara sebesar 13% pada tahun 2017 (Target: $50,77 Juta).
## Tech Stack & Metrics
Tools: Google Spreadsheets (Cleaning), Power BI (Visualization).
KPI Utama: Total Sales (NA), Total Product, Cost Per Click (CPC).
## Data Analysis Methodology
1. Data Cleaning & Preparation (Google Spreadsheets):
- Membersihkan data duplikat dan null values
- mengisi nilai yang hilang (missing values) dengan median pada kolom penting
- mengecek tipe data dan convert ke number
Manual Calculation: Menghitung target pertumbuhan 13% secara manual berdasarkan performa tahun 2016 untuk menetapkan benchmark keberhasilan.
2. Exploratory Data Analysis (EDA):
- Mengidentifikasi tren historis penjualan di Amerika Utara dari tahun 1987 hingga 2020.
- Membedah performa penjualan berdasarkan kategori Rating, Genre, dan Platform untuk menemukan pola produk terlaris.
3. Data Visualization (Power BI):
Menyusun dashboard interaktif menggunakan Power BI untuk memvisualisasikan temuan secara grafis.
## Key Data Insights (North America)
Berdasarkan analisis data historis, berikut adalah profil pemimpin pasar di wilayah Amerika Utara:

| Kategori | Leader | Total Sales (NA) |
| :--- | :--- | :--- |
| **Top Genre** | Action | $863.17 M |
| **Top Rating** | E (Everyone) | $2,494.66 M |
| **Top Platform** | X360 | $595.74 M |
| **Top Publisher** | Nintendo | $815.86 M |

**Temuan Utama:** **Rating "E" (Everyone)** adalah penyumbang pendapatan terbesar. Strategi pemulihan harus difokuskan pada konten ramah keluarga.
## Strategic Recommendations

### **A. Strategi Produksi (Production)**
* **Priority Development:** Memperbanyak produksi game genre **Action** dengan rating **"E"**.
* **Exclusive Growth:** Mengembangkan judul eksklusif pada platform dengan basis massa besar seperti X360.
* **Cycle Acceleration:** Mempercepat siklus pengembangan produk untuk merespons dinamika pasar.

### **B. Pemasaran & Operasional**
* **Influencer Marketing:** Kolaborasi dengan **game streamers** untuk menjangkau audiens muda di NA.
* **Social Media & Bundling:** Kampanye media sosial agresif dan paket **diskon bundling**.
* **CPC Optimization:** Alokasi budget iklan pada channel digital dengan efisiensi biaya terbaik.
* **Distribution:** Optimasi distribusi fisik di area dengan permintaan tinggi di Amerika Utara.
  
## Project Impact
Output proyek ini adalah dashboard strategis yang membantu manajemen memantau progres target **$50,77 Juta** dan memprioritaskan alokasi sumber daya pada kategori yang paling menguntungkan.
